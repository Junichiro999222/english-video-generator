有名な洋楽を使って英語を勉強したい！……けど、ネイティブのスピードや発音が難しくて、なかなかリスニング練習に取り入れづらい、なんて感じたことはありませんか？
そんな悩みを少しでも解消できたらと思い、自分のプログラミングスキルの向上も兼ねて、この動画を作ってみました🎶

この動画のポイント：
　・聞き取りやすい字幕表示（チャンクごとに歌詞タイミングを最適化）
　・意味がスッと入る字幕切替（英語と日本語を交互に表示）
　・耳と頭で理解できる構成（文のまとまり単位で繰り返し練習にも◎）
感想やアドバイスなど、コメントもお待ちしています！


実行環境： AWS EC2（Python 3.10）
実行ツール：
　　・OpenAI Whisper（音声認識）
    ・ChatGPT API（チャンク分割、翻訳、字幕整形、キーワード抽出など）
    ・DALL·E、Pexels API（画像生成・検索）
    ・Voicebox API（英語音声合成）、AWS Polly（日本語音声合成）
    ・FFmpeg、ASS字幕（合成・演出）
　　・Youtube API


大まかな制作フロー：
アーティストを指定 
-（以下、全自動）-
有名曲トップ3を検索 → 指定された曲をダウンロード 
- （以下、各曲について） -
mp3を読み込み → Whisperで文字起こし → 一番を抽出 → 和訳
→ 音声合成 → イメージ生成 → 字幕作成 → FFmpegで動画を合成
→ Youtubeに自動でアップロード
